# 评议报告格式规范

## 报告结构

```
# 评议报告

## 基本信息
- 议案标题：
- 评议时间：
- 参与角色：
- 置信度等级：高 / 中 / 低 / 存疑
- 迭代轮次：

---

## 结论摘要
（一段话，不含"建议采纳与否"）

---

## 支持意见
| 来源角色 | 核心论点 | 置信度 | 依据 |
|---------|---------|-------|------|

---

## 反对 / 风险意见
| 来源角色 | 核心论点 | 严重程度(1-5) | 置信度 | 依据 |
|---------|---------|-------------|-------|------|

---

## 未解决的核心争议
（若有）
- 争议焦点：
- 起草官立场：
- 批驳官立场：
- 无法收敛原因：

---

## 信息缺口
（若有）
- 缺失内容：
- 建议获取方式：

---

## 评议过程摘要
（可折叠，记录辩论轮次与关键转折）
```

---

## 置信度等级说明

| 等级 | 含义 |
|------|------|
| 高 | 各角色基本一致，依据充分 |
| 中 | 存在分歧但主流意见清晰 |
| 低 | 争议较大，结论不稳定 |
| 存疑 | 存在关键信息缺口，结论不可靠 |

---

## 严重程度说明（风险意见）

| 分值 | 含义 |
|------|------|
| 5 | 致命风险，可能导致方案完全失败 |
| 4 | 重大隐患，需要方案调整 |
| 3 | 中等风险，需要专项应对 |
| 2 | 轻微问题，可接受或有替代方案 |
| 1 | 边缘性意见，供参考 |
