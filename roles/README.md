# 角色定义说明

本目录定义了评议系统中每个角色的职责边界、Prompt规范和工具权限。

## 固定角色

| 文件 | 角色 | 层级 |
|------|------|------|
| `00-decomposer.md` | 议案解构官 | 准备层 |
| `01-drafter.md` | 起草官 | 评议层 |
| `02-critic.md` | 批驳官 | 评议层 |
| `03-integrator.md` | 整合官 | 整合层 |

## 动态加载角色

`specialists/` 目录下为专科顾问模板，由准备层根据议案领域动态触发。

新增专科顾问时，复制任意现有模板，修改领域名称和工具列表即可。

## Prompt 规范

每个角色文件包含：
1. **角色定位**：一句话说明职责
2. **行为约束**：明确不允许做什么
3. **输出格式**：结构化输出要求
4. **可调用工具**：该角色有权调用的工具列表
5. **System Prompt 模板**：可直接使用的提示词
